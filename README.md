# 每日心情语录

一个根据心情色彩推荐名言的网页应用。

## 功能特点

- 五种心情色彩选择
- 丰富的名言数据库
- 支持随机切换名言
- 优雅的动画效果
- 响应式设计

## 在线使用

访问 [https://marshallwang07.github.io/daily-quotes](https://marshallwang07.github.io/daily-quotes)

## 本地运行

1. 克隆仓库：
```bash
git clone https://github.com/marshallwang07/daily-quotes.git
```

2. 直接在浏览器中打开 index.html 或使用 VS Code 的 Live Server 插件

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
